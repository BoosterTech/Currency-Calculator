# Currency-calculator

## Converting PLN amounts into different currencies
### Demo:
 https://boostertech.github.io/Currency-calculator/
### How it work?
1. Insert an **Amount** in PLN (required field)
2. Choose **Currency** you would like to convert to
3. Click "**Calculate!**"
