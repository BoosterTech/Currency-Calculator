# Currency-calculator
Converting amounts into different currencies form PLN !
![main picure](https://www.linkpicture.com/q/main_18.png)
## How it works?
1. Insert an **Amount** in PLN (required field)
2. Choose **Currency** you would like to convert to
3. Click "**Calculate**"
